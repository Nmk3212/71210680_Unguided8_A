2001+2002+2003+.........+3998+3999+4000 =

Proses
pertama mencari n dengan menggunakan rumus Un
Un   = a + (n-1)b
4000 = 2001 + (n-1).1
1999 = n - 1
2000 = n
n    = 2000

lalu menggunakan rumus Sn
Sn = n/2 (a + Un)
   = 2000/2 (2001 + 4000)
   = 1000 (6001)
   = 6001000

jadi, 2001+2002+2003+.........+3998+3999+4000 adalah 6001000
